## Overview

**Task Flow** is a backend REST API that helps users manage long-term projects through structured, sequential execution. Rather than a flat task list, it introduces the concept of a **WorkPlan** — a structured workflow where complex projects are decomposed into ordered phases, completed one step at a time.

---

## Features

| Feature | Description |
|---|---|
| **WorkPlan System** | Organize projects into ordered phases with sequential execution |
| **AI-Generated Tips** | Context-aware productivity tips powered by **Groq API** based on task title and description |
| **JWT Authentication** | Secure stateless authentication with **Redis**-backed token management |
| **Task Scheduling** | Flexible time slots — Morning, Afternoon, Evening, Night |
| **Automated Reminders** | Background **Cron Jobs** that monitor deadlines and trigger alerts |
| **API Documentation** | Full **Swagger UI** for interactive endpoint exploration |
